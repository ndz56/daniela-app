# היומן של דניאלה 💜

אפליקציית ווב (PWA) בעברית, ידידותית לטכנופוביה, ליומן יומי - פגישות, תרופות, בדיקות, ימי הולדת, ארוחות שבת והערות.

---

## 🚀 איך מעלים את זה לאוויר - מדריך לדוד (לא מתכנת)

המטרה: שדניאלה תוכל להיכנס מהטלפון שלה לאפליקציה, וגם להתקין אותה במסך הבית כמו כל אפליקציה רגילה.

### האופציה המומלצת: Netlify Drop (חינמי, ללא צורך בידע טכני)

**שלב 1 - צור תיקיית ZIP:**
1. פתח את התיקייה `C:\Users\david\Daniela-App` בסייר הקבצים של חלונות.
2. סמן את כל הקבצים והתיקיות (Ctrl+A) - כולל `index.html`, `app.js`, `styles.css`, `manifest.json`, `sw.js`, ותיקיית `icons`.
3. **חשוב:** אל תסמן את התיקייה כולה, אלא את התוכן שבתוכה. לחץ ימני → "Send to" → "Compressed (zipped) folder".
4. תן לקובץ שם, למשל `daniela-app.zip`.

**שלב 2 - העלאה ל-Netlify Drop:**
1. גש לכתובת: **https://app.netlify.com/drop**
2. גרור את קובץ ה-ZIP לתוך החלון.
3. תוך 30 שניות תקבל קישור (למשל: `https://elegant-hopper-12345.netlify.app`).
4. **שמור את הקישור הזה!** זה הקישור שדניאלה תפתח.
5. אופציונלי: הירשם (חינם) כדי לבחור שם יפה יותר ולשמור את האתר.

**שלב 3 - שלח לדניאלה:**
1. שלח לה את הקישור ב-WhatsApp.
2. שתפתח אותו ב-**Chrome** (לא ב-WhatsApp Browser).
3. בתפריט של כרום (שלוש נקודות) - "התקן את האפליקציה" או "הוסף למסך הבית".
4. עכשיו יש לה אייקון על מסך הבית - תלחץ עליו והאפליקציה תפתח במסך מלא כמו כל אפליקציה.

### עדכונים בעתיד
כשארצה לשנות משהו - תכין ZIP חדש ותגרור שוב לאותו פרויקט ב-Netlify. בלחיצה אחת.

## 🔔 פיצ'רים

- **מסך בית** עם תאריך לועזי + עברי + תקציר היום
- **פגישות** - כולל פגישות חוזרות (שבועי/חודשי/שנתי)
- **תרופות** עם תזכורת אוטומטית בשעה שבחרת
- **בדיקות דם** עם הערות (צום? איפה?)
- **ימי הולדת** - לוח עברי וגם לועזי, תאריך הבא מחושב אוטומטית
- **ארוחת שבת** - בחירה בשרי/חלבי/פרווה עם הצעות תפריט כשרות
- **פתקים אישיים**
- **התראות** - בטלפון תקבל התראה כשמגיע זמן תרופה / 30 דקות לפני פגישה
- **זמני שבת** - הדלקת נרות לפי עיר (תל אביב, ירושלים וכו')
- **חגים** - הצגה אוטומטית של החג הקרוב
- **גיבוי** - יצוא וייבוא קובץ JSON עם כל הנתונים

## ✨ הצעות AI לארוחת שבת - איך מפעילים?

האפליקציה תומכת ב-Claude (Anthropic) כדי לקבל הצעות תפריט דינמיות. אין חובה - יש הצעות מובנות שעובדות בלי כלום. אבל אם רוצים את האפקט "וואו":

### איך להוציא מפתח Anthropic (3 דקות):

1. גש ל-**https://console.anthropic.com**
2. הירשם בכתובת מייל (יש כ-$5 קרדיט חינם בהתחלה - יספיק להמון הצעות)
3. בתפריט הצד: **"API Keys"** → **"Create Key"**
4. תן לו שם ("daniela-app") ולחץ Create
5. **העתק את המפתח** (מתחיל ב-`sk-ant-`). חשוב - הוא יוצג רק פעם אחת!
6. באפליקציה: לחץ על גלגל ההגדרות (⚙) → גלילה ל"הצעות AI לארוחת שבת" → הדבק → שמירה

### כמה זה עולה?
- כל הצעה ≈ $0.003 (פחות מאגורה)
- שימוש אישי טיפוסי: $0.50-$1 לחודש
- חיוב לפי שימוש בפועל, ניתן לקבוע תקרה ב-Anthropic Console

### אבטחה
- המפתח נשמר רק במכשיר של דניאלה (localStorage), לא בשרת
- אם תאבד הטלפון - שני את המפתח ב-Anthropic Console
