☁️ סנכרון בין מכשירים

חיבור לענן Firebase כדי לשתף נתונים בין הטלפון של דניאלה לטלפון שלך.

איך מקבלים את ההגדרות? (5 דקות)
  1. גש ל-console.firebase.google.com (התחבר עם Gmail)
  2. לחץ "Add project" → תן שם (לדוגמה: yoman) → הבא → ביטול Analytics → צור
  3. במסך הראשי לחץ על האייקון </> (Web) → תן שם אפליקציה → רישום
  4. תראה קוד עם firebaseConfig = {...} - העתק רק את החלק בתוך הסוגריים המסולסלים
  5. בתפריט הצד: Build → Firestore Database → Create database → Production mode → בחר אזור (europe-west) → הפעל
  6. בלשונית Rules של Firestore - תחליף הכל ב:
    rules_version='2';
    service cloud.firestore {
      match /databases/{db}/documents {
        match /families/{f} {
          allow read, write: if request.auth != null;
        }
      }
    }     ולחץ Publish
  7. בתפריט הצד: Build → Authentication → Get started → לשונית Sign-in method → Anonymous → Enable → Save
  8. חזור לכאן, הדבק את ה-config בשדה למעלה ולחץ "חיבור לסנכרון"
  9. בטלפון של דניאלה - הזיני את אותו קוד משפחה ואת אותו config - והנתונים יסונכרנו!

✨ הצעות AI לארוחת שבת

הדבק כאן מפתח Anthropic API כדי לקבל הצעות ארוחה דינמיות. אפשר גם בלי - יש הצעות מובנות באפליקציה.

איך מקבלים מפתח? (3 דקות)
  1. גש ל-console.anthropic.com
  2. הירשם בכתובת מייל (יש $5 קרדיט חינם להתחלה)
  3. בתפריט הצד: "API Keys" → "Create Key"
  4. העתק את המפתח (מתחיל ב-sk-ant-) והדבק כאן
  5. עלות מוערכת לשימוש אישי: פחות מ-$1 לחודש

⚠ המפתח נשמר רק במכשיר שלך, לא בשרת. אל תשתף אותו.
